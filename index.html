WikiProject Biodiversity

Tools that connect GBIF, iNaturalist, Plazi and Wikidata to describe and illustrate more taxa on Wikipedia and Wikimedia Commons.

Tools & apps

  • Tarsier

    Find openly-licensed, reusable images of any taxon — across GBIF and the taxonomic literature (Plazi / BLR) — and upload them to Wikimedia Commons in one click.

    Open ↗ Source by Andra Waagmeester

  • iNotWiki

    Command-line tool that finds taxa missing a Wikipedia article by cross-referencing iNaturalist observations with Wikidata.

    GitHub Codeberg by Andra Waagmeester

  • WikiBlitz finder

    Browser app that finds taxa missing a Wikipedia article in one or more languages at once — scoped by taxon, country, iNaturalist project or user, or an OpenStreetMap area — querying GBIF, Wikidata & iNaturalist live. Exports a ready-to-paste worklist.

    Open ↗ Source by Andra Waagmeester

  • Taxon-name stub maker

    Interactive notebook (Voilà / Binder) that guides an editor through drafting an English Wikipedia stub for a taxon from extracted sources.

    GitHub Launch ↗ by Andra Waagmeester

  • inat2ebird

    Converts historical iNaturalist bird observations into an eBird Record Format CSV ready for upload.

    GitHub by Tiago Lubiana

  • iNat2Wiki

    Connects iNaturalist and Wikimedia projects — generates Wikipedia stub articles for taxa from iNaturalist observations (this link: Portuguese).

    Open tool ↗ by Tiago Lubiana
